국어 서술형 답안 AI 채점 보조 웹앱 - 기능 아이디어 정리

1. 입력/촬영
- 사진 다중 업로드 (학급 전체 일괄 처리)
- 비스듬한 사진 자동 보정 (기울기, 조명)
- 답안지 템플릿 인식 (이름/학번/문항 영역 자동 분리)

2. 손글씨 인식 (OCR)
- 손글씨 → 텍스트 변환
- 문맥 기반 단어 추정 (모호한 글자 앞뒤 문맥으로 보정)
- 줄바꿈/문단 구조 유지
- 인식 신뢰도 표시 (글자/문장 단위)
- 판독 불확실 구간 별도 표시
- 원본 이미지 + 변환 텍스트 나란히 대조

3. 채점 기준 설정
- 모범답안 등록
- 핵심 키워드/논점 등록
- 항목별 배점 입력
- 유사 표현(동의어) 허용 설정

4. AI 채점 판단
- 채점 기준별 충족 여부 자동 체크
- 근거 문장 하이라이트 (어떤 표현이 어떤 기준을 만족시켰는지)
- AI 추천 점수 제시 (자동 확정 아님)
- 선생님 최종 점수 확정 단계 분리

5. 교정/검토 인터페이스
- 인식 결과 직접 수정 가능
- 점수 직접 수정 가능
- 피드백 코멘트 초안 생성 (수정 후 사용)

6. 결과 관리
- 학생별 결과 모아보기
- 엑셀/CSV 내보내기
- 자주 미달성된 채점 기준 통계 (수업 보완용)

7. 신뢰성/운영
- 이의제기 시 AI 판단 근거 재확인 기능
- 채점 일관성 로그 (같은 기준 동일 적용 여부)
- 답안 이미지 저장 기간/접근 권한/삭제 정책
- 클라우드 API vs 학교 내부망 처리 방식 결정
